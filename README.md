# AI-Powered E-commerce Web App

## Overview
This project is a simple e-commerce web application built using AI and deployed on Vercel.

The goal was to quickly create, deploy, and customize a web app without managing servers, while using real-world developer workflows.

## Features
- AI-generated UI using v0.dev
- Landing page with hero section
- Product page with Buy Now button
- Checkout page with Stripe placeholder
- Secure HTTPS hosting
- Global CDN and automatic deployments
- Real user performance monitoring with Vercel Speed Insights

## Tools & Technologies
- v0.dev
- Vercel
- GitHub
- Cursor 2.0
- Stripe (UI placeholder)

## How It Works
1. The UI was generated using plain English prompts in v0.dev
2. The project was exported to GitHub
3. Deployment was done using Vercel (one-click deploy)
4. The project was cloned locally and customized using Cursor
5. Changes were pushed to GitHub and auto-deployed
